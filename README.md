# 🕒 Chronos – Task Management Kanban Board

Chronos is a sleek and interactive **Kanban-style task management web app** built using **HTML, CSS, and JavaScript**.
It helps you organize tasks efficiently across different stages: **To Do, In Progress, and Done**.

---

## 🚀 Live Demo

👉 https://chronos-lac.vercel.app/

---

## 📌 Features

* 🧩 **Drag & Drop Tasks**
  Move tasks seamlessly between columns.

* 💾 **Local Storage Persistence**
  Tasks remain saved even after refreshing the page.

* ➕ **Add New Tasks**
  Simple modal-based task creation.

* ❌ **Delete Tasks**
  Remove tasks instantly with a click.

* 🔔 **Toast Notifications**
  Real-time feedback for actions like adding, deleting, and moving tasks.

* 📊 **Dynamic Task Count**
  Each column shows the number of tasks it contains.

* 🎨 **Modern UI/UX**
  Smooth animations, clean layout, and responsive design.
